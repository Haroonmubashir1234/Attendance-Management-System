**Attendance Management System**

**Project Overview**

The Attendance Management System is a console-based application developed in C language to manage student attendance in educational institutions. It replaces manual attendance registers with an automated system that improves accuracy, efficiency, and record management. The system includes three modules: Student, Teacher, and Exam Department. Each module has its own login system and functionalities such as attendance marking, record viewing, and exam eligibility checking.

**Features**

**Student Module**

Student registration and login
View attendance records
Check attendance percentage

**Teacher Module**

Teacher registration and login
Add students
Mark, update, and delete attendance
Search student records
View data files

**Exam Department Module**

Secure login system
Check student eligibility for exams
View student and attendance records

**Technologies Used**

C Programming Language
Structures and Arrays
Pointers
Functions
File Handling (Text and Binary)
String Handling

**Files Used**

students.txt – Stores student records
teachers.txt – Stores teacher records
exam.txt – Stores exam department data
attendance.dat – Binary file for attendance storage
attendance.txt – Human-readable attendance report

**How to Run**

Compile main.c
Run the program
Follow the menu system to use the application
